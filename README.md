## 📖 Descripción del Proyecto

Este aplicativo web es una solución integral para la **gestión de finanzas personales**, diseñada para simplificar la administración de ingresos, egresos y deudas mediante una interfaz intuitiva y automatizada.

El núcleo del sistema incorpora **Inteligencia Artificial** para potenciar la toma de decisiones financieras. A través de un **Chatbot inteligente**, los usuarios pueden registrar transacciones en lenguaje natural (ej: *"Gasté 50k en gasolina hoy"*), eliminando la fricción de los formularios manuales. Además, el sistema ofrece herramientas de **proyección financiera**, permitiendo a los usuarios establecer metas de ahorro a corto, mediano y largo plazo, calculando la viabilidad y el tiempo necesario para alcanzarlas basándose en sus hábitos de consumo históricos.

### ✨ Características Principales

*   **Registro Conversacional:** Ingreso de gastos e ingresos mediante un Chatbot con procesamiento de lenguaje natural.
*   **Dashboard Financiero:** Visualización clara del balance actual, desglose de gastos y estado de deudas.
*   **Proyecciones Inteligentes:** Algoritmos que estiman tiempos de ahorro para metas específicas (viajes, compras, fondo de emergencia).
*   **Gestión de Deudas:** Seguimiento de obligaciones pendientes y planificación de pagos.
